# UTS Pemrograman Web - CRUD Mahasiswa

Aplikasi web sederhana berbasis PHP Native untuk mengelola data mahasiswa sebagai tugas Ujian Tengah Semester.

## Fitur Utama
* **Create:** Menambah data mahasiswa beserta unggah foto profil.
* **Read:** Menampilkan daftar mahasiswa dalam tabel yang rapi.
* **Update:** Mengubah informasi mahasiswa dan memperbarui foto.
* **Delete:** Menghapus data mahasiswa.
* **Validation:** Validasi sisi klien menggunakan JavaScript (cek kosong, tipe file, dan ukuran file max 2MB).

## Teknologi yang Digunakan
* HTML & CSS
* PHP Native
* MySQL Database
* JavaScript

## Cara Instalasi
1. Clone repository ini ke folder `htdocs` atau `www`.
2. Import file `db_mahasiswa.sql` ke phpMyAdmin.
3. Sesuaikan konfigurasi database di `koneksi.php`.
4. Jalankan melalui browser `localhost/uts-pemrograman-web`.
